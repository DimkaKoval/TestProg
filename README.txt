Створити Web аплікацію для парсінгу та зберігання даних в базу даних.
 
1. На головній сторінці користувач бачить кнопку для вибору текстового файлу та поле для вводу 
слова для пошуку. Також нижче має відображатись весь список раніше збережених речень.
 
2.Вибравши текстовий файл та ввівши слово для пошуку система розпізнає та зберігає в базі даних 
всі речення в яких хоча б один раз зустрічається слово для пошуку. Речення потрібно зберігати з 
буквами у зворотньому напрямку, зберігаючи кількість входжень слова в даному реченні. 
Реченням вважається набір слів від крапки до крапки.

 
Технології: .Net (та супутні)
